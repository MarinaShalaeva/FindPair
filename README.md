# Игра «Найди пару»

Автор проекта — Шалаева Марина.

Для написания игры использовался движок Unreal Engine 4 версии 4.27.

В случае попытки скачать архив с проектом, некоторые файлы будут отображены некорректно, так как при загрузке данных в репозиторий использовался Git LFS (Large File Storage). В связи с этим корректное отображение данных на локальном устройстве возможно только при клонировании репозитория.

Для того чтобы корректно открыть проект на локальном устройстве в IDE, необходимо нажать правой кнопкой мыши на файл **FindPair.uproject**, а затем нажать на строчку **Generate Visual Studio project files**.

При необходимости можно обновить проект до более поздней версии, также щелкнув правой кнопкой мыши по файлу **FindPair.uproject** и выбрав строчку **Switch Unreal Engine version...**
